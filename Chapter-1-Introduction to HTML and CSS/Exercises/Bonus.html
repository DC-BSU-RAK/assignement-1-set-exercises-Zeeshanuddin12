HTML Quick Referance

10 HTML tags

  • html-Root element of an HTML document.
  • head-Contains meta data, links to styles, scripts, etc.
  • body-Contains the main content of the webpage.
  • h1 to h6-Headings (from largest, h1, to smallest, h6).
  • p-Helps you write Paragraph.
  • a-Anchor (use for hyperlink).
  • image-Helps you with uploading image.
  • Div-For making boxes
  • span-Inline container for text or elements
  • ul/ol/li-(ul) is for unordered list, (ol) is for ordered list, (li) is for list item

CSS Quick Referance

08 CSS tags

  • color-for adding colour to text.
  • background-color-for changing the background colour.
  • font-sizefor changing the font size.
  • margin-for spacing between elements.
  • paddingfor creating space between elements, border and condent
  • border- for creating border around elements.
  • font-family- choose the font type.
  • display- for how a element is displayed (example:- block, inline, flex, etc.).
